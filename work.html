Mock Landing Page - Monarch Butterflies

This is the ground-breaking, first-ever project I have coded. The landing page is to grab your attention, leaving you wanting more. The subject of butterflies is something I am passionate about. I hope to teach you a little and lead you to find out more by clicking on the working links to legit websites. These links take you to websites that either provide you more information or allow you to donate to the cause. The "Learn More" button provides an alert. Your email address will not be stored and the request for additional information will not be sent, as this is a Frontend project only at this point. Perhaps at a later date, the Backend will be created. Also, it is responsive to a point - Bootstrap was not introduced into this project. That may be another addition to making this first project even better.

This project is built with HTML5, CSS3 and Javascript. The Mock Landing Page is hosted on Netlify and open-sourced on GitHub.

Vanilla Js Weather App

Ever needed to check the weather? This app will check the weather for the city you type into the search. By choosing the current button, weather conditions are displayed for your current location using longitude and latitude. This app will provide current conditions, a 3-hour increment temperature for checking the rest of the day and a 4-day outlook. Temperatures can be in either Celsius or Fahrenheit measures. It will also provide the date and time.

This app is built using HTML5, CSS3, Vanilla Javascript, APIs, Bootstrap, Geo Location functions and Celsius/Fahrenheit Conversion functions. It is hosted on Netlify and open-sourced on GitHub.

Launch React Weather App

This isn't just another weather app. It is built entirely different. It is built using React for a more manageable coding experience. This app will check the weather for the city you type into the search. It will provide date, time, current conditions, an hourly incremented temperature for checking the rest of the day and a 5-day outlook. Animating weather icons have been added. The main temperature can be in either Celsius or Fahrenheit measures. I am currently learning Redux. This app will be improved upon by adding state management by implementing Redux. Check back to see the update.

This app was built using HTML5, CSS3, JSX, React, APIs, Bootstrap, Loops, States and Celsius/Fahrenheit Conversion functions. It is hosted on Netlify and open-sourced on GitHub.

React Dictionary App

Increase your vocabulary. What does the word mean and are there multiple definitions? How do you pronounce it? And how is it used? The answers can be found with this app. Type in a word and take in all the information displayed through its API. There are even visuals to help convey the definition.

This app is built with HTML5, CSS3, JSX, React, APIs, Bootstrap, and it has an Audible element to hear how to pronounce the word. It is hosted on Netlify and open-sourced on GitHub.
